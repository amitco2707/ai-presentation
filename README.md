# AI for Beginners — From Zero to Hero

An interactive, web-based presentation built as a single-page React app. Premium animations via Framer Motion, design via Tailwind CSS.

## 🌐 View the live presentation

**[https://amitco2707.github.io/ai-presentation/](https://amitco2707.github.io/ai-presentation/)**

No install required — open in any browser.

## Navigation

- **Scroll wheel** or **↓ / ↑** / **PgDn / PgUp** / **Space** — move between slides
- **Home / End** — jump to first / last
- **Right-side dots** — click any slide
- **Bottom arrows** — next / previous

## Slide order

1. Title — animated mascot, particles
2. The Basics — AI and ML defined
3. The Hierarchy — the Russian Doll visual
4. Neural Networks — animated node graph
5. Medical Imaging — real-world example
6. LLM vs Agentic — "talker" vs "doer"
7. Glossary — 7 must-know terms
8. Claude Code — simulated terminal
9. Capabilities — Skills, Multi-agent, Plugins
10. Model Ecosystem — LLM / Claude Code / Cowork
11. Outro — welcome to the era
